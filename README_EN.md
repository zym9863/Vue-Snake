## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Mode

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Game Controls

- **Arrow Keys** - Control snake movement direction
- **Space Bar** - Pause/Resume game
- **R Key** - Restart game

- Vue 3 - Progressive JavaScript Framework
